# Division

The `Division` API calculates the quotient of two numbers.

## Parameters

* `num1`: The dividend
* `num2`: The divisor

## Example Usage

```python  
division = Division()  
result = division.fn(10, 2)  
print(result)  # Output: 5.0  
```

Note: This API raises a ZeroDivisionError if the divisor is zero.
